## Instructions
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

Install Stripe Packages
```bash
npm install @stripe/react-stripe-js @stripe/stripe-js
```

Start Payment after running the main.tsx
```bash
python app.py
```

To deactivate server:
```bash
deactivate
```
